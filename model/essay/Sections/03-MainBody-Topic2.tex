\subsection{Topico 2: segundo aspecto do tema}

\topicclaim{Explique aqui a ideia central deste segundo topico.}

Este topico deve acrescentar uma nova etapa ao argumento. Ele pode aprofundar
uma consequencia, apresentar uma comparacao, discutir uma causa ou analisar uma
limitacao relacionada ao tema principal.

Use transicoes para mostrar a relacao com o topico anterior. Por exemplo:
``alem disso'', ``por outro lado'', ``como consequencia'' ou ``em contraste''.
